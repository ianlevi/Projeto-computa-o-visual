# Projeto: Equalização de Histograma (SDL3)

Cumpre: execução por CLI, detecção se a imagem já é cinza, conversão (Y=0.2125R+0.7154G+0.0721B), duas janelas (principal centralizada + painel acoplado), histograma, média e desvio com classificação, equalização toggle (botão/tecla E), salvar com S.

## Requisitos
- MSYS2 UCRT64 (Windows)
- Pacotes:
  - mingw-w64-ucrt-x86_64-gcc
  - mingw-w64-ucrt-x86_64-sdl3
  - mingw-w64-ucrt-x86_64-sdl3-image
  - mingw-w64-ucrt-x86_64-sdl3-ttf (opcional para texto)

## Compilar
```bash
make
```

## Rodar
```bash
./app.exe assets/lena.png
```

### Controles
- **E** / botão: Equalizar ↔ Original
- **S**: Salvar `output_image.png`
- **Esc**: Sair
